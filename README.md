# TruckPark

Aplicação em Node.js para gerenciamento de serviços, como cadastro de clientes e ordens de serviço.  
Possui backend estruturado e funcionalidades para facilitar o controle operacional.

## Tecnologias Utilizadas

- Node.js  
- MongoDB  
- Electron  
- jsPDF

## Funcionalidades

- CRUD completo para clientes e ordens de serviço  
- Geração de relatórios em PDF  
- Interface desktop com Electron  
- Validações e prompts interativos

## Como Rodar o Projeto

1. Clone o repositório:  
   `git clone https://github.com/kayqueFranco/TruckPark.git`
2. Instale as dependências:  
   `npm install`
3. Configure o MongoDB localmente ou na nuvem  
4. Execute o projeto:  
   `npm start`
